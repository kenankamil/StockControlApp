# StockControlApp

Sql Script dosyası açılır. Run etmeden önce en üst satırda yer alan **USE** [ProductDb] GO alanı yerine **CREATE** [ProductDb] GO yazılır.Script Sql Server 2019 versiyonu ile alınmıştır. Alt versiyonlarda çalışmama ihtimali mevcut.

Proje Kenan.CodeBaseCodeChallange.UI projesi ile çalıştırılır. appsetting.json içerisindeki server bağlantı adresini değiştirmek zorunda olabilirsiniz.
